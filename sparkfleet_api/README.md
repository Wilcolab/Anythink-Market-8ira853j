# SparkFleet Predictive Maintenance API (Smart Meeting Assistant)

This project automates meeting transcription, summarization, and action item tracking for SparkFleet. It integrates with calendars, GitHub Issues, and optionally Slack, supporting user workflows and persona adaptation.

## Features
- Automated transcription & summarization
- Task & issue tracking
- Integration points (GitHub, calendar, Slack)
- User workflow (approval, dashboard, etc.)
- Persona adaptation
- Clarity & feedback UI
- Non-functional requirements: performance, security, data retention, reliability, usability, scalability, compliance

See `spec.md` for full requirements and acceptance criteria.
